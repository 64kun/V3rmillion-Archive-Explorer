# V3rmillion Archive Explorer
**V3rmillion** - крупный форум, существовавший около 12 лет. Он был посвящён скриптам, эксплойтам и техническому сообществу вокруг игры `Roblox`. Правда, к сожалению, форум был закрыт в 2023 году. Насколько я помню, на его закрытие повлияли эти факторы:
- Администрация устала поддерживать сайт (банить скаммеров; постоянно удалять вирусы, которые выкладывались пользователями: например, фишинг под видом "Free Robux generator", вредоносные Luau скрипты, RAT, стиллеры, майнеры; модерировать миллионы сообщений и т.д.)
- Был запущен новый Roblox античит `Hyperion` от Byfron Technologies
- Разработчики `Synapse X` (крупнейшего эксплойта для Roblox на тот момент) ушли работать в Roblox

В тот год произошло достаточно, чтобы всё начало рушиться. Я говорю это как тот, кто был очевидцем происходящего, и как тот, кого те события хоть как-то касались. Правда, скажу честно, я никогда не был участником сообщества v3rmillion. Поэтому, прошу прощения, если кому-то не понравится, что я решил сделать этот проект.

Подробнее про v3rmillion вы можете узнать [здесь](https://logos.fandom.com/wiki/V3rmillion) или [здесь](https://theprogamers-experiment.fandom.com/wiki/V3rmillion).

Как я выше уже написал, форум был закрыт в 2023 году. Однако, был опубликован [архив](https://archive.org/download/v3rmillion) со всеми тредами и пользователями этого сообщества. Единственная проблема, решение которой я не смог найти - пользоваться этим архивом невозможно. Вот несколько причин:
1. Невозможно найти нужные треды по названию (нет системы поиска)
2. html страницы каждого из треда ссылаются на v3rmillion - их нельзя открыть
3. Читать эти html страницы просматривая их исходный код очень трудно

Поэтому, я решил написать Explorer, который поможет искать, извлекать все посты, а также строить более понятную историю ответов (да, на удивление, в v3rmillion это было сделано плохо).

**Почему вообще кому-то нужны старые посты с удалённого форума?**
- Там осталось большое количество тредов, которые имеют огромную ценность - можно банально отыскать гайд на что-то, чему не научит тот же ChatGPT, и в интернете вы это легко не найдёте или не найдёте вовсе. Даже сейчас не существует ни одного форума, на котором вы могли бы найти хотя бы 5% той информации, что осталась на v3rmillion.
- История. Да, кто-то может захотеть узнать, "А как было раньше?" - на v3rmillion всё есть.

# Download
**Предварительно созданный поисковый индекс будет загружен автоматически при первом запуске main.py.**

Выберете папку, в которую хотите скачать проект, затем откройте в ней терминал и введите эту команду:
```sh
git clone https://github.com/64kun/V3rmillion-Archive-Explorer.git
```

# Usage
Перейдите в папку с проектом:
```sh
cd V3rmillion-Archive-Explorer
```

Простой пример использования:
```sh
python main.py путь-к-папке-с-тредами-архива-v3rmillion -r "request 1"
```

Вы можете указать максимальное количество результатов (если не укажете, то будет установлено значение по умолчанию: 10):
```sh
python main.py путь-к-папке-с-тредами-архива-v3rmillion -r "request 1" 20
```

Также можно искать сразу по нескольким запросам:
```sh
python main.py путь-к-папке-с-тредами-архива-v3rmillion -r "request 1" "request 2" 15 "request 3" 5
```

После выполнения скрипта все результаты сохранятся в папку `parsed/` в проекте.


*На данный момент поддерживается только сохранение тредов в json формат. В планах сделать генерацию собственных html, но пока неизвестно, когда у меня появится на это время.*
